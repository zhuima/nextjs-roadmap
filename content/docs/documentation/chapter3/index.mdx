这一章为初学者概述了在开始使用 Next.js 时要考虑的基本方面。它强调先决条件，例如理解 JavaScript 和 React，以及对 API 和数据库的熟悉。本章鼓励读者探索 Next.js 文档，该文档是了解其特性和功能的综合资源。此外，它还强调了与充满活力的 Next.js 社区互动并利用教程和课程等在线资源的重要性。本章还重点介绍了需要注意的常见陷阱，包括服务器端渲染和静态站点生成之间的差异、Next.js 内置 API 路由的限制以及样式的细微差别。通过注意这些方面，初学者可以通过 Next.js 充分利用他们的学习之旅。
