## 为什么选择 Next.js?

最好的 Next.js 教程，是 [官方文档](https://nextjs.org/docs)，读它就够了。我们完全没兴趣从头到尾写一本 Next.js 编程教材，不仅因为官方写得足够详细和通俗易懂，而且它就放在那里。 虽然你在官方网站上就是很难找到它的中文版，但是，我们建议你就只看英文版，因为离开了这个教程之后，还是要面对绝大多数都是英文的现实。

本站当前由追马和章鱼两个人维护，如果你也对 Next.js 感兴趣，欢迎和我们一起共同建设.

我最近完成了一个长达数月的工作项目，将多租户 Web 应用程序从 Create React App 转换为 Next.js。我们很高兴在测试版结束后采用该 app 目录，但与此同时，我想通过创建一个个人博客网站来获得一些关于新应用程序路由器的个人经验。

Next.js 以其多功能性，易用性，强大的功能以及与 Vercel 服务的紧密集成而闻名。Next.js 提供服务器端呈现、静态站点生成和 API 路由处理程序，使其成为现代博客网站的理想选择。

## 为什么要沉淀这本手册？

这本书的目的在于将经验凝聚成一本工作手册。每当我们完成一项工作梳理或活动复盘，我们便整理出这样一本手册，或实体的纸质版，或电子的数字版。这些手册也常常成为教学课程的附赠品，赠予学员。当朋友们目睹数十本手册时，他们纷纷惊讶不已。他们渐渐发现，原来经验可以凝聚成这样一本手册！

倾听我对于制作工作手册的五个好处的分享吧，让我们激励彼此行动起来：

1.**【内部晋升】**：将自身工作经验化作册页，分享给团队或公司内部。上司们会沉思，你是那个可塑之才，此刻，晋升的可能因你而生。

2.**【面试通过】**：带上两册手册走向面试，即时展示自己从前的经历与案例，面试官则将题目你更高的几率。

3.**【谈判利器】**：我时常携带几册手册展开合作洽谈。有时需要，展现而言，仿佛在他们眼中，你锋芒毕露，可靠专业。于是，合作接踵而至。

4.**【学员福利】**：作为一位教师，我们所当做的，就是“尽力让学员偷懒”。他们需要得到上手指导，立即动手。因此，为他们派送实施手册作参考，成为一种理想之选，使他们能快速实践。

5.**【成就获得】**：多年工作经验得以凝聚成册，你将汇聚满满的成就感。若是你知识储备尚未足够写书，何妨先制作手册。

在技能的成长之路上，唯求及时凝聚，方可连续累积。将经验总结，心存虚怀学习更多知识。勇往前行，勇于挑战，方能冲破极限，拥有他人无法企及的竞争力。
